In javascript, each variable has a memory location and a placeholder that holds the value of the variable.
Undefined Variables: So, if a varible is created, and is not given a value, then such variable is called undefined.
In these variables, the placeholder has the keyword undefined.
Which means that whenever we try to print out the variable, it shows undefined.
Not-defined variables: If a variable isn't created in javascript, and we try to print out the variable,
Then we get an error in the debug console called the specific variable is not defined.
It means that, when we try to print out the variable that is not even created,
Then it is called not-defined variable.
